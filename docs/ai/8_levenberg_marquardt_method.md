## 1. 位置づけ

[ガウス・ニュートン法](./7_gauss_newton_method.md) には 2 つの故障モードがあった。

1. 線形化の有効範囲を超えて**飛びすぎ**、目的関数がかえって増える
2. $J^\top J$ が特異・悪条件だと更新量 $\boldsymbol{\delta}$ が**解けない・暴れる**

Levenberg-Marquardt 法 (LM 法) は、この両方を**ダンピング**という 1 つの仕掛けで同時に解決する。非線形最小二乗法の実務における事実上の標準アルゴリズムであり、このリポジトリのロードマップの終着点である。

## 2. 更新式

ガウス・ニュートン法の正規方程式にダンピング項 $\lambda I$ ($\lambda > 0$) を足す。

$$
( J^\top J + \lambda I ) \, \boldsymbol{\delta} = - J^\top \mathbf{r}
$$

たったこれだけの変更で、次が保証される。

- **常に解ける。** $J^\top J$ は半正定値なので、$\lambda > 0$ なら $J^\top J + \lambda I$ は正定値 (固有値がすべて $\lambda$ 以上) となり、$J$ がランク落ちしていても可逆。
- **常に下り方向。** 係数行列が正定値なので $\boldsymbol{\delta}^\top \nabla E = -2 \boldsymbol{\delta}^\top ( J^\top J + \lambda I ) \boldsymbol{\delta} < 0$。

Marquardt の提案した変種では、$\lambda I$ の代わりに $\lambda \operatorname{diag}(J^\top J)$ を使う。パラメータごとのスケールの違い (単位の違いなど) をダンピングが自動で吸収するため、スケーリングの悪い問題に強い。以下では簡単のため $\lambda I$ で説明する。

## 3. 2 つの極限 — ガウス・ニュートンと最急降下の補間

$\lambda$ を動かすと、LM 法は既に学んだ 2 つの手法の間を連続的に移り変わる。

- **$\lambda \to 0$**: 更新式はガウス・ニュートン法 $J^\top J \boldsymbol{\delta} = - J^\top \mathbf{r}$ に一致する。
- **$\lambda$ が大きいとき**: $J^\top J$ が $\lambda I$ に比べて無視でき、$\boldsymbol{\delta} \approx - \frac{1}{\lambda} J^\top \mathbf{r} = - \frac{1}{2 \lambda} \nabla E$。これは[最急降下法](./5_steepest_descent.md)の小さな一歩である ($\lambda$ が大きいほど歩幅が縮む)。

つまり $\lambda$ は「ガウス・ニュートン法をどれだけ信用するか」のつまみである。

> うまく進めているときは $\lambda$ を小さくしてガウス・ニュートン法の**速さ**を使い、モデルが信用できないときは $\lambda$ を大きくして最急降下法の**頑健さ**に退避する。

「終盤に速いが遠方で危うい手法」と「遅いが必ず下る手法」の切り替えを、$\lambda$ という 1 つの実数の増減だけで実現しているのが LM 法の設計の妙である。

## 4. 信頼領域としての解釈

ダンピングにはもう 1 つ、より現代的な解釈がある。ガウス・ニュートン法の部分問題に「線形化が信用できる範囲から出ない」という制約を付けた

$$
\min_{\boldsymbol{\delta}} \| \mathbf{r} + J \boldsymbol{\delta} \|^2
\qquad \text{s.t.} \quad \| \boldsymbol{\delta} \| \leq \Delta
$$

を考える (**信頼領域**問題)。この制約付き最小化の解は、ある $\lambda \geq 0$ に対する

$$
( J^\top J + \lambda I ) \, \boldsymbol{\delta} = - J^\top \mathbf{r}
$$

の解と一致することが示せる ($\lambda$ は制約のラグランジュ乗数で、制約がきついほど大きくなる)。つまり

> LM 法の $\boldsymbol{\delta}$ は、「線形化モデルを信頼できる半径 $\Delta$ の球の中で、モデルを最小化する一歩」であり、$\lambda$ を増やすことは信頼半径を狭めることに対応する。

「$\lambda$ はステップ幅を抑える正則化パラメータ」という素朴な理解と、「$\lambda$ は信頼領域の半径の裏返し」という幾何的理解の両方を持っておくとよい。なお、$J^\top J + \lambda I$ という形は線形最小二乗のリッジ正則化 (チホノフ正則化) と同型であり、[SVD](./3_singular_value_decomposition.md) の言葉では「小さい特異値 $\sigma_j$ による誤差増幅 $1 / \sigma_j$ を $\sigma_j / ( \sigma_j^2 + \lambda )$ に抑える」操作になっている。悪条件への耐性はここから来る。

## 5. $\lambda$ の適応更新

$\lambda$ は固定せず、反復のたびに「線形化モデルの予測がどれだけ当たったか」で調整する。判定には**ゲイン比**

$$
\rho = \frac{E(\boldsymbol{\beta}) - E(\boldsymbol{\beta} + \boldsymbol{\delta})}{L(\mathbf{0}) - L(\boldsymbol{\delta})}
\qquad
\bigl( L(\boldsymbol{\delta}) = \| \mathbf{r} + J \boldsymbol{\delta} \|^2 \text{: 線形化モデルの予測値} \bigr)
$$

を使う。分子が「実際の減少量」、分母が「モデルが予測した減少量」で、分母は更新式を使うと $L(\mathbf{0}) - L(\boldsymbol{\delta}) = \boldsymbol{\delta}^\top ( \lambda \boldsymbol{\delta} - J^\top \mathbf{r} )$ と計算でき、常に正である。

- $\rho$ が 1 に近い: 線形化はよく当たっている → 一歩を採択し、$\lambda$ を減らして次はもっとガウス・ニュートン寄りに
- $\rho$ が小さい・負: モデルの予測が外れた ($E$ が増えた) → 一歩を**棄却**し、$\lambda$ を増やして小さく安全な一歩でやり直す

全体のアルゴリズム (Madsen–Nielsen 流の更新則) は次のようになる。

```text
入力: 初期値 β, τ (例 1e-3), 停止閾値 ε₁ ε₂, 最大反復数
r, J を計算; A = JᵀJ; g = Jᵀr
λ = τ · max(diag(A)); ν = 2
repeat:
    (A + λI) δ = −g を解く
    if ‖δ‖ ≤ ε₂ (‖β‖ + ε₂): 収束 (更新量が微小)
    β_new = β + δ
    ρ = (E(β) − E(β_new)) / (δᵀ(λδ − g))
    if ρ > 0:                       # 目的関数が減った → 採択
        β = β_new
        r, J, A, g を再計算
        if ‖g‖∞ ≤ ε₁: 収束 (勾配が微小)
        λ = λ · max(1/3, 1 − (2ρ − 1)³);  ν = 2
    else:                           # 増えた → 棄却してやり直し
        λ = λ · ν;  ν = 2ν
```

棄却時に $\nu$ を倍々にするのは、「連続で失敗しているならもっと大胆に $\lambda$ を上げる」ため。$\lambda$ が十分大きくなれば最急降下法の微小ステップになり、いつかは必ず $E$ が減るので、この繰り返しが無限に続くことはない。

## 6. 実装上の注意

- **$J^\top J$ を作らずに解ける。** 各反復の方程式は、行列を縦に積んだ線形最小二乗問題

$$
\min_{\boldsymbol{\delta}}
\left\|
\begin{bmatrix} J \\ \sqrt{\lambda} \, I \end{bmatrix} \boldsymbol{\delta}
-
\begin{bmatrix} - \mathbf{r} \\ \mathbf{0} \end{bmatrix}
\right\|^2
$$

  と等価である (正規方程式を作ると $( J^\top J + \lambda I ) \boldsymbol{\delta} = - J^\top \mathbf{r}$ に戻ることを確認せよ)。これを [QR 分解](./2_qr_decomposition.md) で解けば、条件数を二乗させずに済む。

- **ヤコビ行列の入手方法。** 解析的に導出するのが最良 (速く正確)。導出が困難なら数値微分 (実装が楽だが誤差と評価回数に注意) や自動微分を使う。ヤコビ行列の符号や転置の取り違えは LM 法の定番バグなので、数値微分との突き合わせで検証する価値がある。
- **停止条件は複数併用する。** 勾配の微小 ($\| J^\top \mathbf{r} \|_\infty \leq \varepsilon_1$)、更新量の微小、最大反復数の 3 つを併用するのが定石。
- **保証されるのは局所解まで。** LM 法は各反復で $E$ を減らすが、たどり着くのは初期値の谷の底である。大域的最小の保証はなく、初期値の質が結果を左右する事実は[非線形最小二乗法](./4_nonlinear_least_squares.md)の注意のまま変わらない。

## 7. ロードマップ全体のまとめ

LM 法は、このリポジトリで積み上げてきた部品の集大成である。

| トピック | LM 法での役割 |
| --- | --- |
| 1. 線形最小二乗法 | 各反復で解く部分問題そのもの (正規方程式) |
| 2. QR 分解 | 部分問題を条件数を悪化させずに解く道具 |
| 3. SVD | 悪条件・ダンピングの効果を理解する視点 (特異値と誤差増幅) |
| 4. 非線形最小二乗法 | 問題設定と $\nabla E = 2 J^\top \mathbf{r}$ という構造 |
| 5. 最急降下法 | $\lambda$ 大の極限。頑健さの供給源 |
| 6. ニュートン法 | 速さの理想形。$H + \tau I$ の正定値化がダンピングの原型 |
| 7. ガウス・ニュートン法 | $\lambda \to 0$ の極限。速さの供給源 |

> Levenberg-Marquardt 法とは、残差の線形化 (ガウス・ニュートン法) で得た線形最小二乗の部分問題にダンピング $\lambda I$ を加え、モデルの当たり外れ (ゲイン比) を見ながら $\lambda$ を増減することで、ガウス・ニュートン法の速さと最急降下法の頑健さを自動で切り替える手法である。

ここまで理解できれば、あとは実装で確かめるだけである。ゼロ残差に近い問題で二次収束的な速さを、悪い初期値で $\lambda$ が自動的に持ち上がって粘る様子を、それぞれ観察できれば、このロードマップは完走と言ってよい。
