## 1. この文書で学ぶこと

[5. 最急降下法](./5_steepest_descent.md) では、負の勾配方向に進む反復法を学んだ。実装は簡単で確実に下れるが、収束は一次 (誤差が毎回一定の比率でしか減らない) であり、その比率は条件数 $\kappa$ に支配されるのだった。実際、サンプルコードの実験では $f = (x^2 + 1000 y^2)/2$ という単純な 2 次関数ですら、$\kappa = 1000$ のために約 7,000 回の反復を要した。

この文書で学ぶ**ニュートン法 (Newton's method)** は、同じ問題を**たった 1 回の反復**で解く。魔法ではない。最急降下法が勾配 (1 階微分) しか見ないのに対し、ニュートン法は**曲率 (2 階微分)** も見る。谷がどちらにどれだけ細長いかを知った上で底へ飛ぶから速いのである。

話の流れは次の通り。

1. 準備として、関数を**2 次式で近似する**方法を 1 変数で導入する (§2)
2. まず方程式 $g(x) = 0$ の解を求める**求根のニュートン法**を学び、「正しい桁数が毎回倍になる」二次収束を $\sqrt{2}$ の計算で体感する (§3)
3. それを最小化問題に拡張する。1 変数で「2 次関数で近似して頂点へ跳ぶ」を導出し (§4)、多変数では 2 階偏微分を並べた**ヘッセ行列**が登場する (§5)
4. 2 次関数なら 1 反復で厳密解に着くこと、つまり条件数の呪いから解放されることを確かめる (§6)
5. 最後に弱点と、その対策である $H + \tau I$ 修正を見る。これは後の [Levenberg-Marquardt 法](./8_levenberg_marquardt_method.md) の原型になる (§7)

前提知識は [0. 数学の準備](./0_math_preliminaries.md) の偏微分・勾配・行列・凸関数である。テイラー展開やヘッセ行列は前提にしない。必要な分だけこの文書内で導入する。

## 2. 準備 — 関数を 2 次式で近似する

### 2.1 1 次近似 = 接線 (復習)

微分可能な関数 $f$ の点 $x$ における接線は

```math
y = f(x) + f'(x) h
\qquad (h = \text{$x$ からのずれ})
```

である。$h$ が小さいうちは、曲線は接線とほとんど区別がつかない。すなわち

```math
f(x + h) \approx f(x) + f'(x) h
```

これを **1 次近似**と呼ぶ。「その点での値と傾きが一致する直線」で関数を置き換えている。

### 2.2 2 次近似 — 曲がり具合まで真似る

直線は曲がれないので、1 次近似は関数の「曲がり具合」を全く表現できない。そこで、直線の代わりに 2 次式 (放物線)

```math
q(h) = a + b h + c h^2
```

で近似することを考える。係数 $a, b, c$ は「$h = 0$ において、値・傾き・曲がり具合 (2 階微分) の 3 つが $f$ と一致する」ように決める。順に条件を書くと

```math
q(0) = a = f(x), \qquad
q'(0) = b = f'(x), \qquad
q''(0) = 2c = f''(x)
```

3 つ目の条件から $c = \frac{1}{2} f''(x)$ である ($q''(h) = 2c$ に注意。**係数に $\frac{1}{2}$ が付くのは、$h^2$ を 2 回微分すると 2 が出てくるから**である)。まとめると

```math
f(x + h) \approx f(x) + f'(x) h + \frac{1}{2} f''(x) h^2
```

これを **2 次近似**と呼ぶ。この文書全体を支える式なので、具体例で確かめておこう。$f(x) = x^3$ を $x = 1$ の周りで近似する。$f(1) = 1$、$f'(x) = 3x^2$ より $f'(1) = 3$、$f''(x) = 6x$ より $f''(1) = 6$ なので

```math
(1 + h)^3 \approx 1 + 3h + 3h^2
```

一方、左辺を展開すると $(1+h)^3 = 1 + 3h + 3h^2 + h^3$ だから、**近似の誤差はちょうど $h^3$** である。$h = 0.1$ なら誤差は $0.001$、$h = 0.01$ なら $0.000001$。$h$ を 10 分の 1 にすると誤差は 1000 分の 1 になる。1 次近似の誤差が $h^2$ 程度 (この例では $3h^2 + h^3$) だったのに比べ、一段と速く消える。

> [!NOTE]
> この「値・1 階微分・2 階微分・… が一致する多項式で関数を近似する」考え方には**テイラー展開**という名前が付いており、何次まででも続けられる。この文書では 2 次まであれば十分である。

## 3. 求根のニュートン法 — 接線で根を探す

### 3.1 アイデア: 接線を引いて x 軸との交点へ移る

ニュートン法はもともと、非線形方程式

```math
g(x) = 0
```

の解 (**根**と呼ぶ) を求める手法である。$g$ が 2 次方程式なら解の公式があるが、一般の $g$ に解の公式はない。そこで次のように考える。

> 曲線 $y = g(x)$ そのものと $x$ 軸の交点は直接求められない。しかし**直線**と $x$ 軸の交点なら一瞬で求められる。ならば、今いる点で曲線を**接線で置き換え**、その接線と $x$ 軸の交点へ移ればよい。移った先でまた接線を引き、また交点へ移る。これを繰り返せば、根に近づくにつれ曲線と接線の差は消えていくから、交点は本物の根に吸い寄せられていくはずである。

頭の中に絵を描いてほしい。曲線上の点 $(x_k, g(x_k))$ から接線を滑り降りて $x$ 軸にぶつかった場所が次の点 $x_{k+1}$ である。そこから真上 (または真下) の曲線上の点へ移り、また接線を滑り降りる。根の近くでは曲線がほぼ直線に見えるので、この「滑り降り」は驚くほど正確に根を言い当てる。

### 3.2 更新式の導出

絵を式にする。点 $x_k$ における $y = g(x)$ の接線は、§2.1 の 1 次近似そのもので

```math
y = g(x_k) + g'(x_k)(x - x_k)
```

である。この接線が $x$ 軸と交わる点、つまり $y = 0$ となる $x$ を求める。

```math
0 = g(x_k) + g'(x_k)(x - x_k)
```

$g(x_k)$ を移項して

```math
g'(x_k)(x - x_k) = -g(x_k)
```

$g'(x_k) \neq 0$ として両辺を $g'(x_k)$ で割ると

```math
x - x_k = -\frac{g(x_k)}{g'(x_k)}
```

よって、次の点は

```math
x_{k+1} = x_k - \frac{g(x_k)}{g'(x_k)}
```

これが**求根のニュートン法の更新式**である。「現在の関数値 $g(x_k)$ を傾き $g'(x_k)$ で割った分だけ戻る」と読める。関数値が大きければ大きく動き、傾きが急ならば少しだけ動く。理にかなっている。

### 3.3 √2 を手で計算する

$g(x) = x^2 - 2$ の正の根は $\sqrt{2} = 1.41421356237\ldots$ である。$g'(x) = 2x$ なので更新式は

```math
x_{k+1} = x_k - \frac{x_k^2 - 2}{2 x_k}
= \frac{2 x_k^2 - (x_k^2 - 2)}{2 x_k}
= \frac{x_k^2 + 2}{2 x_k}
= \frac{1}{2} \left( x_k + \frac{2}{x_k} \right)
```

と整理できる。「$x_k$ と $2 / x_k$ の平均を取る」という美しい形である ($\sqrt{2}$ は $x$ と $2/x$ がちょうど一致する点なので、両者の平均を取れば真ん中へ寄る、と直観的にも読める)。

$x_0 = 2$ から手で回してみる。

```math
x_1 = \frac{1}{2} \left( 2 + \frac{2}{2} \right) = \frac{3}{2} = 1.5
```

```math
x_2 = \frac{1}{2} \left( \frac{3}{2} + \frac{2}{3/2} \right)
= \frac{1}{2} \left( \frac{3}{2} + \frac{4}{3} \right)
= \frac{1}{2} \cdot \frac{17}{6}
= \frac{17}{12} = 1.41666\ldots
```

2 回の手計算ですでに 3 桁合っている。続きはサンプルコード `src/bin/6_newton_method.rs` の実行結果 (倍精度浮動小数点) で見る。下線は誤差が始まる桁である。

| $k$ | $x_k$ | 誤差 $x_k - \sqrt{2}$ | 正しい桁数 (目安) |
| --- | --- | --- | --- |
| 0 | $2.000000000000000$ | $+5.858 \times 10^{-1}$ | 0 桁 |
| 1 | $1.5\underline{00000000000000}$ | $+8.579 \times 10^{-2}$ | 1 桁 |
| 2 | $1.41\underline{6666666666667}$ | $+2.453 \times 10^{-3}$ | 2〜3 桁 |
| 3 | $1.41421\underline{5686274510}$ | $+2.124 \times 10^{-6}$ | 5〜6 桁 |
| 4 | $1.41421356237\underline{4690}$ | $+1.595 \times 10^{-12}$ | 11〜12 桁 |
| 5 | $1.414213562373095$ | 倍精度で誤差ゼロ | 15 桁以上 |

誤差の指数に注目してほしい: $10^{-1} \to 10^{-2} \to 10^{-3} \to 10^{-6} \to 10^{-12}$。指数が**ほぼ倍々**で深くなっている。言い換えると、**正しい桁数が反復のたびにおよそ 2 倍になる**。1 桁、3 桁、6 桁、12 桁…という増え方であり、5 回目で倍精度浮動小数点の限界 (約 16 桁) に到達して打ち止めになった。反復 1 回の中身は「割り算 1 回と平均 1 回」程度なのだから、恐るべき効率である。

### 3.4 なぜ倍々なのか — 厳密な計算

この例では、誤差の振る舞いを近似なしで厳密に計算できる。誤差を $e_k = x_k - \sqrt{2}$ とおく。更新式から

```math
x_{k+1} - \sqrt{2}
= \frac{1}{2} \left( x_k + \frac{2}{x_k} \right) - \sqrt{2}
```

右辺を通分する。

```math
= \frac{x_k^2 + 2 - 2 \sqrt{2}\, x_k}{2 x_k}
```

分子は $x_k^2 - 2\sqrt{2}\, x_k + (\sqrt{2})^2$ であり、これは完全平方式である。

```math
= \frac{(x_k - \sqrt{2})^2}{2 x_k}
```

すなわち

```math
e_{k+1} = \frac{e_k^2}{2 x_k}
```

**次の誤差は今の誤差の 2 乗に比例する**。$x_k \approx \sqrt{2}$ なので比例定数は $\frac{1}{2 \sqrt{2}} \approx 0.354$ であり、たとえば $e_k = 10^{-3}$ なら $e_{k+1} \approx 0.354 \times 10^{-6}$。誤差 $10^{-6}$ の次は $10^{-12}$。表で見た倍々の正体はこれである。

一般の $g$ でも同じことが起きる。根 $x^*$ の周りで $g$ を 2 次近似 (§2.2) すると

```math
0 = g(x^*) \approx g(x_k) + g'(x_k)(x^* - x_k) + \frac{1}{2} g''(x_k)(x^* - x_k)^2
```

一方、ニュートン法の更新式を変形すると $g(x_k) = g'(x_k)(x_k - x_{k+1})$ なので、これを代入して

```math
0 \approx g'(x_k)(x_k - x_{k+1}) + g'(x_k)(x^* - x_k) + \frac{1}{2} g''(x_k)(x^* - x_k)^2
```

最初の 2 項をまとめると $g'(x_k)(x^* - x_{k+1})$ になるから

```math
x_{k+1} - x^* \approx \frac{g''(x_k)}{2 g'(x_k)} (x_k - x^*)^2
```

つまり誤差はやはり 2 乗で減り、比例定数は $\left| \frac{g''}{2 g'} \right|$ 程度である ($g(x) = x^2 - 2$ なら $g'' = 2$、$g' = 2x$ で、先ほどの厳密な計算と一致する)。この導出から、**根で $g'(x^*) \neq 0$ であること**と、**2 次近似が通用する程度に根の近くから始めること**が条件だと分かる。

### 3.5 二次収束 — 桁数の言葉で

いま観察した性質を一般的な言葉にしておく。誤差 $e_k = x_k - x^*$ が、ある定数 $C > 0$ に対して

```math
|e_{k+1}| \leq C |e_k|^2
```

を満たすとき、この反復は**二次収束 (quadratic convergence)** するという。

この式の意味は「桁数」に翻訳すると腑に落ちる。正しい桁数を $d_k = -\log_{10} |e_k|$ とおく (誤差 $10^{-3}$ なら $d_k = 3$)。上の不等式の両辺の $-\log_{10}$ を取ると

```math
d_{k+1} \geq 2 d_k - \log_{10} C
```

つまり**正しい桁数が毎回およそ 2 倍になる** ($-\log_{10} C$ は定数のずれで、桁数が増えるほど相対的に無視できる)。$\sqrt{2}$ の表で桁数が $1 \to 2.6 \to 5.7 \to 11.8$ と推移したのは、まさに $d_{k+1} \approx 2 d_k + 0.45$ ($C \approx 0.354$ なので $-\log_{10} C \approx 0.45$) である。

対比のために、最急降下法の**一次収束** $|e_{k+1}| \leq r |e_k|$ ($0 < r < 1$) を同じ言葉で書くと

```math
d_{k+1} \geq d_k + (-\log_{10} r)
```

桁数が毎回**一定量ずつ足し算で**増える。しかも [5. 最急降下法](./5_steepest_descent.md) で見た通り $r$ は条件数に支配され、$\kappa = 100$ では 1 桁増やすのに約 60 反復かかる。**足し算 (しかも歩幅は条件数次第) と倍々のかけ算**。これが一次収束と二次収束の違いであり、ニュートン法の最大の魅力である。

## 4. 最適化への拡張 — 1 変数

### 4.1 最小化は求根に化ける

ここからが本題である。求めたいのは方程式の根ではなく、関数 $f$ の最小点だとする。滑らかな関数の最小点では接線が水平になる、つまり

```math
f'(x) = 0
```

が成り立つ (このような点を**停留点**と呼ぶ)。ならば、$g(x) = f'(x)$ とおいて**求根のニュートン法をそのまま適用**すればよい。§3.2 の更新式で $g \to f'$、$g' \to f''$ と置き換えて

```math
x_{k+1} = x_k - \frac{f'(x_k)}{f''(x_k)}
```

これが**最適化のニュートン法** (1 変数版) である。収束の速さも求根のときと同じで、解の近くでは二次収束する。

### 4.2 もう 1 つの導出 — 2 次近似の頂点へ跳ぶ

同じ式には、最適化の文脈でより直観的な導出がある。§2.2 の 2 次近似

```math
f(x_k + h) \approx q(h) = f(x_k) + f'(x_k) h + \frac{1}{2} f''(x_k) h^2
```

を考える。$q(h)$ は $h$ の 2 次関数、つまり放物線である。**本物の $f$ の最小点は分からなくても、放物線の最小点 (頂点) なら平方完成で厳密に求められる**。そこで「$f$ をその点での放物線で置き換え、放物線の頂点へ一気に跳ぶ」ことにする。

頂点を求める。$q$ を $h$ で微分して 0 とおくと

```math
q'(h) = f'(x_k) + f''(x_k) h = 0
```

```math
h = -\frac{f'(x_k)}{f''(x_k)}
```

よって跳んだ先は $x_{k+1} = x_k + h = x_k - \frac{f'(x_k)}{f''(x_k)}$ となり、§4.1 と**完全に同じ式**が出てくる。平方完成で確認してもよい。$f''(x_k) > 0$ のとき

```math
q(h) = \frac{1}{2} f''(x_k) \left( h + \frac{f'(x_k)}{f''(x_k)} \right)^2 + f(x_k) - \frac{f'(x_k)^2}{2 f''(x_k)}
```

であり、確かに $h = -f'(x_k)/f''(x_k)$ で最小になる。

まとめると、ニュートン法には 2 つの顔がある。

> - 求根の顔: $f' = 0$ という方程式を「接線で置き換えて」解く
> - 最適化の顔: $f$ を「放物線で置き換えて」その頂点へ跳ぶ
>
> この 2 つは同じ式である。1 次近似 ($f'$ の接線) と 2 次近似 ($f$ の放物線) は、微分を 1 回挟んだだけの同じ情報だからである。

### 4.3 伏線 — $f'' < 0$ だと何が起きるか

平方完成の式をもう一度見てほしい。$f''(x_k) < 0$ なら放物線は**上に凸**であり、「頂点」は最小点ではなく**最大点**である。それでも更新式は同じ場所に跳ぶので、ニュートン法は**山を登ってしまう**。$f' = 0$ は極小点だけでなく極大点でも成り立つから、求根の顔から見ても「間違った根」に向かい得る。これは §7 で扱う弱点の核心なので、頭の片隅に置いておいてほしい。

## 5. 多変数化 — ヘッセ行列

### 5.1 偏微分をもう一回並べる

多変数関数 $f : \mathbb{R}^m \to \mathbb{R}$ に話を広げる。[0. 数学の準備](./0_math_preliminaries.md) の通り、1 階の偏微分を縦に並べたものが勾配

```math
\nabla f =
\begin{pmatrix}
\dfrac{\partial f}{\partial x_1} \\
\vdots \\
\dfrac{\partial f}{\partial x_m}
\end{pmatrix}
```

だった。1 変数の $f''$ に対応するものは何か。勾配の各成分は再び $m$ 変数の関数なので、**それぞれをもう一回、各変数で偏微分できる**。結果は $m \times m$ 個あるから、表 (行列) に並べる。

```math
H =
\begin{pmatrix}
\dfrac{\partial^2 f}{\partial x_1 \partial x_1} & \cdots & \dfrac{\partial^2 f}{\partial x_1 \partial x_m} \\
\vdots & \ddots & \vdots \\
\dfrac{\partial^2 f}{\partial x_m \partial x_1} & \cdots & \dfrac{\partial^2 f}{\partial x_m \partial x_m}
\end{pmatrix}
```

この行列を**ヘッセ行列 (Hessian matrix)** と呼び、$H(\mathbf{x})$ または $\nabla^2 f(\mathbf{x})$ と書く。$(i, j)$ 成分は「$x_j$ で偏微分してから $x_i$ で偏微分したもの」である。

具体例で手を動かす。$f(x, y) = x^2 + xy + 2y^2$ とすると

```math
\nabla f =
\begin{pmatrix}
2x + y \\
x + 4y
\end{pmatrix}
```

第 1 成分 $2x + y$ を $x$ で偏微分すると $2$、$y$ で偏微分すると $1$。第 2 成分 $x + 4y$ を $x$ で偏微分すると $1$、$y$ で偏微分すると $4$。並べて

```math
H =
\begin{pmatrix}
2 & 1 \\
1 & 4
\end{pmatrix}
```

$(1,2)$ 成分と $(2,1)$ 成分がどちらも $1$ になったのは偶然ではない。滑らかな関数では偏微分の順序を入れ替えても結果が変わらない ($\frac{\partial^2 f}{\partial x \partial y} = \frac{\partial^2 f}{\partial y \partial x}$、シュワルツの定理) ため、**ヘッセ行列は常に対称行列** ($H^\top = H$) である。

### 5.2 多変数の 2 次近似

1 変数の 2 次近似 $f(x + h) \approx f + f' h + \frac{1}{2} f'' h^2$ の多変数版は、$f' \to \nabla f$、$f'' \to H$、積 → 内積・2 次形式と置き換えた

```math
f(\mathbf{x}_k + \boldsymbol{\delta})
\approx
f(\mathbf{x}_k) + \nabla f(\mathbf{x}_k)^\top \boldsymbol{\delta} + \frac{1}{2} \boldsymbol{\delta}^\top H(\mathbf{x}_k) \, \boldsymbol{\delta}
```

である ($\boldsymbol{\delta}$ が「ずれ」を表すベクトルで、1 変数の $h$ に当たる)。2 変数で成分を書き下すと、$\boldsymbol{\delta} = (a, b)^\top$ として

```math
f(x + a, \, y + b)
\approx
f + \frac{\partial f}{\partial x} a + \frac{\partial f}{\partial y} b
+ \frac{1}{2} \left(
\frac{\partial^2 f}{\partial x^2} a^2
+ 2 \frac{\partial^2 f}{\partial x \partial y} a b
+ \frac{\partial^2 f}{\partial y^2} b^2
\right)
```

1 変数の式と見比べれば「各方向の 1 次の項 + 各方向ペアの 2 次の項」という自然な拡張になっていることが分かる。

これも具体例で検算できる。先ほどの $f(x, y) = x^2 + xy + 2y^2$ を点 $(1, 1)$ の周りで近似してみる。$f(1,1) = 4$、$\nabla f(1,1) = (3, 5)^\top$、$H$ は上で求めた定数行列である。右辺は

```math
4 + 3a + 5b + \frac{1}{2}(2a^2 + 2ab + 4b^2)
= 4 + 3a + 5b + a^2 + ab + 2b^2
```

一方、左辺を直接展開すると

```math
f(1+a, \, 1+b) = (1+a)^2 + (1+a)(1+b) + 2(1+b)^2
= 4 + 3a + 5b + a^2 + ab + 2b^2
```

**完全に一致する**。近似ではなく等号である。これは偶然ではなく、$f$ 自身が 2 次式だからである。**2 次式を 2 次式で近似すれば厳密に一致する** — この当たり前の事実が、§6 で「2 次関数なら 1 反復」という劇的な結果を生む。

### 5.3 2 次モデルの最小化 — ニュートン法の更新式

1 変数のとき (§4.2) と同じ戦略を取る。本物の $f$ の代わりに、2 次近似で作った**局所 2 次モデル**

```math
q(\boldsymbol{\delta}) = f(\mathbf{x}_k) + \nabla f(\mathbf{x}_k)^\top \boldsymbol{\delta} + \frac{1}{2} \boldsymbol{\delta}^\top H(\mathbf{x}_k) \, \boldsymbol{\delta}
```

の最小点へ跳ぶ。1 変数では放物線の頂点だったものが、多変数では**放物面の底**になる。

$q$ の最小点は $\nabla_{\boldsymbol{\delta}}\, q = \mathbf{0}$ を解けばよい。2 変数で愚直に計算する。$\nabla f = (g_1, g_2)^\top$、$H = \begin{pmatrix} h_{11} & h_{12} \\ h_{12} & h_{22} \end{pmatrix}$、$\boldsymbol{\delta} = (\delta_1, \delta_2)^\top$ とおくと

```math
q(\delta_1, \delta_2) = f + g_1 \delta_1 + g_2 \delta_2
+ \frac{1}{2} \left( h_{11} \delta_1^2 + 2 h_{12} \delta_1 \delta_2 + h_{22} \delta_2^2 \right)
```

$\delta_1$、$\delta_2$ それぞれで偏微分すると

```math
\frac{\partial q}{\partial \delta_1} = g_1 + h_{11} \delta_1 + h_{12} \delta_2,
\qquad
\frac{\partial q}{\partial \delta_2} = g_2 + h_{12} \delta_1 + h_{22} \delta_2
```

これを縦に並べると、きれいに行列の形になる。

```math
\nabla_{\boldsymbol{\delta}}\, q = \nabla f + H \boldsymbol{\delta}
```

(一般の $m$ 変数でも、$H$ が対称であることを使って同じ式が成り立つ。) これを $\mathbf{0}$ とおけば、ニュートン法の更新量 $\boldsymbol{\delta}_k$ が満たすべき方程式が得られる。

```math
\boxed{\;
H(\mathbf{x}_k) \, \boldsymbol{\delta}_k = - \nabla f(\mathbf{x}_k),
\qquad
\mathbf{x}_{k+1} = \mathbf{x}_k + \boldsymbol{\delta}_k
\;}
```

これが**最適化のニュートン法** (多変数版) である。1 変数の $x_{k+1} = x_k - f'/f''$ と見比べると、「$f''$ で割る」が「$H$ の連立 1 次方程式を解く」に化けている。逆行列で書けば $\boldsymbol{\delta}_k = -H^{-1} \nabla f$ だが、[2. QR 分解](./2_qr_decomposition.md) で述べたのと同じ理由で、実装では逆行列を作らず連立 1 次方程式として解く。

ここで 1 つ用語を定義する。対称行列 $H$ が、すべての $\boldsymbol{\delta} \neq \mathbf{0}$ に対して

```math
\boldsymbol{\delta}^\top H \boldsymbol{\delta} > 0
```

を満たすとき、$H$ は**正定値 (positive definite)** であるという。これは 1 変数の「$f'' > 0$」の多変数版であり、2 次モデル $q$ がどの方向に切っても下に凸、つまり**お椀型の放物面**であることを意味する。$H$ が正定値のときに限り、$\nabla_{\boldsymbol{\delta}}\, q = \mathbf{0}$ の解は本当に $q$ の最小点である (正定値でない場合に何が起きるかは §7 で扱う)。先ほどの例 $H = \begin{pmatrix} 2 & 1 \\ 1 & 4 \end{pmatrix}$ は

```math
\boldsymbol{\delta}^\top H \boldsymbol{\delta} = 2a^2 + 2ab + 4b^2 = a^2 + (a + b)^2 + 3b^2 > 0
\qquad ((a,b) \neq (0,0))
```

なので正定値である。

### 5.4 最急降下法との違いを言葉にする

- **最急降下法**: $\mathbf{x}_{k+1} = \mathbf{x}_k - \alpha \nabla f$。勾配の向きに進む。歩幅 $\alpha$ は自分で決めるしかない。
- **ニュートン法**: $\mathbf{x}_{k+1} = \mathbf{x}_k - H^{-1} \nabla f$。勾配を曲率で「補正」した向きに、曲率が教える距離だけ進む。歩幅の調整が要らない。

最急降下法のジグザグ ([5. 最急降下法](./5_steepest_descent.md) §4) を思い出そう。等高線が細長い楕円のとき、等高線に垂直な方向 ($-\nabla f$) は谷底の方向とほとんど一致しないのだった。ニュートン法は $H$ によって「この方向は谷が急、この方向は緩やか」という情報を織り込むので、細長い谷でも**底の方向へまっすぐ**飛ぶ。ジグザグは起きない。

## 6. 2 次関数なら 1 反復 — 条件数からの解放

### 6.1 手計算: κ = 1000 の谷を一撃で

最急降下法を苦しめた問題をそのままぶつけてみる。

```math
f(x, y) = \frac{x^2 + 1000 y^2}{2}
```

等高線は $x$ 方向に細長い楕円で、条件数は $\kappa = 1000$ ([2. QR 分解](./2_qr_decomposition.md) で導入した条件数と同じもの)。最小点は明らかに原点である。勾配とヘッセ行列は

```math
\nabla f = \begin{pmatrix} x \\ 1000 y \end{pmatrix},
\qquad
H = \begin{pmatrix} 1 & 0 \\ 0 & 1000 \end{pmatrix}
```

初期値 $(1, 1)$ でニュートン法の方程式 $H \boldsymbol{\delta} = -\nabla f$ を立てると

```math
\begin{pmatrix} 1 & 0 \\ 0 & 1000 \end{pmatrix}
\begin{pmatrix} \delta_1 \\ \delta_2 \end{pmatrix}
=
- \begin{pmatrix} 1 \\ 1000 \end{pmatrix}
```

第 1 行は $\delta_1 = -1$、第 2 行は $1000 \delta_2 = -1000$ より $\delta_2 = -1$。よって

```math
\mathbf{x}_1 = \begin{pmatrix} 1 \\ 1 \end{pmatrix} + \begin{pmatrix} -1 \\ -1 \end{pmatrix} = \begin{pmatrix} 0 \\ 0 \end{pmatrix}
```

**1 回で厳密な最小点に着いた**。$1000$ という係数は方程式の左辺と右辺で打ち消し合っており、$\kappa$ がいくつであろうと結果は変わらない。

### 6.2 なぜ 1 回で終わるのか

理由は §5.2 の検算で既に見た。**$f$ 自身が 2 次関数なら、2 次モデル $q$ は近似ではなく $f$ そのもの**である。だから「$q$ の最小点へ跳ぶ」は「$f$ の最小点へ跳ぶ」と同じであり、どこから始めても 1 反復で厳密解に着く。最急降下法の遅さの原因だった条件数は、そもそも登場する幕がない。

サンプルコード `src/bin/6_newton_method.rs` の実験 2 がこれを確認している。[5. 最急降下法](./5_steepest_descent.md) のサンプルコードと突き合わせると

| 手法 | $\kappa = 1000$ での反復回数 |
| --- | --- |
| 最急降下法 (バックトラッキング付き) | 6,987 回 |
| ニュートン法 | **1 回** |

同じ問題、同じ初期値である。テストコードでは $\kappa = 10^6$ でも 1 回であることを確認している。

もう 1 つ重要な対応がある。[1. 最小二乗法](./1_least_squares_method.md) の目的関数 $E(\boldsymbol{\beta})$ はまさに $\boldsymbol{\beta}$ の 2 次関数だった。したがって**線形最小二乗法にニュートン法を適用すると 1 反復で厳密解に着き、その 1 反復で解いている連立 1 次方程式は正規方程式に他ならない**。「ニュートン法 1 発 = 正規方程式を解くこと」である。閉形式解と反復法がここでつながる。

### 6.3 一般の関数では — 局所二次収束

$f$ が 2 次関数でなければ、2 次モデルは近似に戻り、1 回では終わらない。しかし解 $\mathbf{x}^*$ に近づくほど 2 次モデルは正確になっていくから、終盤は求根のときと同じ**二次収束**、すなわち桁数の倍々が現れる。サンプルコードの実験 3 は、滑らかな凸関数

```math
f(x, y) = e^{x+y-1} + e^{x-y-1} + e^{-x-1}
```

を初期値 $(-1, 1)$ から解いたものである。勾配のノルム $\|\nabla f\|$ (0 に近いほど解に近い) の履歴は

| 反復 $k$ | $\|\nabla f(\mathbf{x}_k)\|$ |
| --- | --- |
| 0 | $6.6 \times 10^{-1}$ |
| 1 | $1.8 \times 10^{-1}$ |
| 2 | $2.4 \times 10^{-2}$ |
| 3 | $1.9 \times 10^{-4}$ |
| 4 | $3.3 \times 10^{-8}$ |
| 5 | $4.5 \times 10^{-16}$ |

序盤 (反復 0〜2) は 2 次モデルがまだ粗く、減りは緩やかである。しかし解に近づいた反復 2 以降は指数が $-2 \to -4 \to -8 \to -16$ と**きれいに倍々**になり、5 反復で倍精度の限界に達した (得られた解は $(-\frac{\ln 2}{2}, 0) \approx (-0.34657, 0)$ で、$\nabla f = \mathbf{0}$ を手計算で解いた厳密解と一致する)。

この「解の近くで二次収束」が成り立つ条件をまとめておく。

- $H(\mathbf{x}^*)$ が正定値である (1 変数の $f''(x^*) > 0$ に相当)
- $H$ が $\mathbf{x}^*$ の近くで急変しない (技術的にはリプシッツ連続と呼ばれる条件)
- 初期値 $\mathbf{x}_0$ が $\mathbf{x}^*$ に十分近い

3 つ目の「十分近い」が実用上の急所であり、次節の主題である。とはいえ、**終盤の追い込みに関してニュートン法より速い実用手法はほぼ存在しない**ことは強調しておきたい。

## 7. 弱点と修正 — H + τI

### 7.1 三つの弱点

強力さと引き換えに、ニュートン法には無視できない弱点がある。

1. **ヘッセ行列が高くつく。** $m$ 変数なら 2 階偏微分が $m^2$ 個 (対称性を使っても約 $m^2/2$ 個) 必要で、導出も実装も面倒である。さらに連立 1 次方程式を解くのに $O(m^3)$ の計算がかかる。$m$ が数個なら問題ないが、$m$ が数千・数百万 (機械学習の典型) では 1 反復すら実行できない。
2. **解から遠いと方向すら信用できない。** §4.3 の伏線を回収する。解から遠い場所では $H$ が正定値とは限らない。$H$ が正定値でないとき、2 次モデルはお椀型ではなく、山型 (上に凸) や**鞍型** (ある方向には下に凸、別の方向には上に凸。その中心を**鞍点**と呼ぶ) になる。このとき $\nabla_{\boldsymbol{\delta}}\, q = \mathbf{0}$ の解は「底」ではなく山頂や鞍点の位置であり、そこへ跳ぶ $\boldsymbol{\delta}$ は**下り方向ですらない**ことがある。また、$\nabla f = \mathbf{0}$ は極大点や鞍点でも成り立つので、反復がそうした点に収束してしまうこともあり得る。
3. **飛びすぎる。** 2 次近似が信用できるのは $\mathbf{x}_k$ の近くだけである ($x^3$ の例で、$h$ が大きいと誤差 $h^3$ が爆発したことを思い出そう)。モデルの有効範囲を超えて大きく跳ぶと、着地点で $f$ がむしろ増えていることがある。

### 7.2 対策

実用では素のニュートン法をそのまま使うことは少なく、対策を組み合わせる。

**直線探索付きニュートン法。** 方向はニュートン法の $\boldsymbol{\delta}_k$ を使い、距離は $\mathbf{x}_{k+1} = \mathbf{x}_k + \alpha_k \boldsymbol{\delta}_k$ として [5. 最急降下法](./5_steepest_descent.md) §3 のバックトラッキングで制御する。弱点 3 (飛びすぎ) への対処である。

**修正ニュートン法 ($H + \tau I$)。** 弱点 2 (正定値でない $H$) への対処。$H$ をそのまま使う代わりに、単位行列の $\tau > 0$ 倍を足した

```math
(H + \tau I) \, \boldsymbol{\delta} = - \nabla f
```

を解く。なぜこれで直るのか。任意の $\boldsymbol{\delta} \neq \mathbf{0}$ に対して

```math
\boldsymbol{\delta}^\top (H + \tau I) \, \boldsymbol{\delta}
= \boldsymbol{\delta}^\top H \boldsymbol{\delta} + \tau \, \boldsymbol{\delta}^\top \boldsymbol{\delta}
= \boldsymbol{\delta}^\top H \boldsymbol{\delta} + \tau \| \boldsymbol{\delta} \|^2
```

であり、第 2 項は $\tau$ に比例していくらでも大きくできる。つまり **$\tau$ を十分大きく取れば、$H$ がどんな行列でも $H + \tau I$ は正定値にできる**。正定値なら 2 次モデルは必ずお椀型になり、$\boldsymbol{\delta}$ は必ず下り方向になる。

さらに $\tau$ の大小は手法の性格を連続的に変える。

- $\tau \to 0$: 純粋なニュートン法。解の近くで速い。
- $\tau$ が大きいとき: $H$ が $\tau I$ に比べて無視でき、$\boldsymbol{\delta} \approx -\frac{1}{\tau} \nabla f$。これは**最急降下法の小さな一歩**である。遅いが必ず下る。

つまり $\tau$ は「ニュートン法をどれだけ信用するか」を調節するつまみであり、**速いが危ういニュートン法と、遅いが頑健な最急降下法を 1 つの実数で橋渡し**している。

> [!IMPORTANT]
> この「$H$ に $\tau I$ を足して安全にする」という発想を必ず覚えておいてほしい。[8. Levenberg-Marquardt 法](./8_levenberg_marquardt_method.md) の更新式 $(J^\top J + \lambda I) \boldsymbol{\delta} = -J^\top \mathbf{r}$ に現れる**ダンピング項 $\lambda I$ は、この修正ニュートン法の $\tau I$ がそのまま原型**である。ここまで理解していれば、LM 法は「知っている話の組み合わせ」として読める。

弱点 1 (ヘッセ行列のコスト) には別の答えがある。目的関数が最小二乗形 $E = \sum_i r_i^2$ という特別な構造を持つ場合、2 階偏微分を一切計算せずに $H$ の良い近似が作れる。それが次の [7. ガウス・ニュートン法](./7_gauss_newton_method.md) である。

## 8. サンプルコード

`src/bin/6_newton_method.rs` に、この文書の内容をそのまま実装してある。

```sh
cargo run --bin 6_newton_method   # 実験の実行
cargo test --bin 6_newton_method  # テストの実行
```

- `newton_root`: 求根のニュートン法 $x_{k+1} = x_k - g(x_k)/g'(x_k)$。実験 1 で $\sqrt{2}$ を計算し、§3.3 の表の数値 (誤差の指数が倍々で減る) を出力する。
- `newton_optimize`: 最適化のニュートン法。$H(\mathbf{x}) \boldsymbol{\delta} = -\nabla f(\mathbf{x})$ を連立 1 次方程式として解いて $\mathbf{x} \mathrel{+}= \boldsymbol{\delta}$ を繰り返し、$\|\nabla f\| \leq \varepsilon$ で停止する。
- 実験 2: 2 次関数 $f = (x^2 + 1000 y^2)/2$ が 1 反復で解けること (§6.1)。
- 実験 3: 滑らかな凸関数での局所二次収束 (§6.3 の $\|\nabla f\|$ の表)。
- テストでは、二次収束の定義式 $|e_{k+1}| \leq C |e_k|^2$ ($C \approx 0.35$) が数値的に成り立つことや、$\kappa = 10^6$ でも 1 反復で解けることも確認している。

なお、このサンプルは弱点への対策 (直線探索や $H + \tau I$) を実装していない「素のニュートン法」である。扱う問題が凸で初期値も穏当なため成立しているが、実験 3 の初期値をわざと遠くに変えてみると、発散したり `solve_linear` が失敗したりする様子が観察できるはずである。

## 9. まとめ

> ニュートン法は「関数をその点での 2 次近似 (放物面) で置き換え、その底へ一気に跳ぶ」手法である。解の近くでは正しい桁数が毎回およそ 2 倍になる二次収束を達成し、$f$ が 2 次関数なら条件数に関係なく 1 反復で厳密解に着く。ただし 2 階微分のコストと、解から遠い場所での不安定さという代償があり、実用には直線探索や $H + \tau I$ による正定値化などの補強が要る。

- 求根 ($g = 0$) と最適化 ($\nabla f = \mathbf{0}$) の 2 つの顔を持つ。どちらも「非線形なものを 1 次モデル (接線) / 2 次モデル (放物面) で置き換えて解く」という同じ思想である。
- 二次収束 $|e_{k+1}| \leq C |e_k|^2$ は、桁数の言葉では $d_{k+1} \approx 2 d_k$、つまり「正しい桁数の倍々」である。一次収束の「桁数の足し算」との差は反復を重ねるほど開く。
- ヘッセ行列 $H$ は「偏微分をもう一回並べた表」であり、1 変数の $f''$ の多変数版。正定値 ($\boldsymbol{\delta}^\top H \boldsymbol{\delta} > 0$) が「下に凸のお椀」の条件である。
- 最急降下法とちょうど相補的: 遠方で頑健だが条件数に苦しむ最急降下法、近傍で圧倒的に速いが遠方で危ういニュートン法。$H + \tau I$ の $\tau$ はこの両者を橋渡しするつまみであり、[8. Levenberg-Marquardt 法](./8_levenberg_marquardt_method.md) のダンピングの原型である。
- 非線形最小二乗法には、2 階微分を計算せずにヘッセ行列を近似できる特別な構造がある。それが次の [7. ガウス・ニュートン法](./7_gauss_newton_method.md) である。
